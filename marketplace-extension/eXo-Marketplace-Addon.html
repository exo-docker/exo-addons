Marketplace Add-on - eXo Platform



eXo Market Place extension helps you to manage your own catalog of Apps or Add-ons, as a real Market Place, on top of eXo Platform. This is a multi-module maven project. See the README.MD file for more information.

You can modify the sources in the left panel and try building the addon in codenvy.


Default usernames and password:

Normal user:    username: exo / password: password
Administrative user:    username: admin / password: password
When you are ready to deploy it and run in eXo platform, click the RUN Button in the toolbar:

The following steps will take place automatically:
The Marketplace Add-on will be built using maven in the runner environment.
The Marketplace Add-on will be deployed to the runner's eXo instance
The runner's eXo instance will be started and you'll be able to test your addon.
